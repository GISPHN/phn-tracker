# 要件定義

## 目的

奈良県を対象に、災害時・訓練時に保健師個人がどこにいて、どのチームに所属し、どのように移動しているかを本部が地図上で把握できるプロトタイプを作る。

## MVP

- 表示名は `自治体名-姓`
- 個人単位で位置を送信
- チーム名は固定候補ではなく自由入力
- 表示チームは、実際に記録されたチーム名から自動生成
- 位置更新の標準値は1分
- 更新間隔は `30秒`, `1分`, `3分`, `5分`, `手動のみ`
- Android / iPhone / PC / タブレット対応
- メモを位置と時刻に紐づける
- CSV / GeoJSON / SVGMap向けSVGを出力
- 初期MVPでは住民個人情報を扱わない

## 本部ビュー

- `本部モード` に切り替えると、訓練セッションID、アクセスコード、訓練作成、共有URLを表示する
- Googleスプレッドシート + Apps Scriptで数十秒ごとに同期する
- 同じ訓練セッションIDとアクセスコードのデータだけを表示する

## 地図

- MapLibre GLで実装
- `Find_LngLat` と同じ `OSMFJ POI`, `OSM Bright`, `全国最新写真` のMapLibreスタイルを利用
- 国土地理院の陰影起伏図を最下層に固定
- 上層地図は半透明にし、奈良県の盆地・山地の起伏を見やすくする
- Overture Maps Places互換GeoJSONをPOIとして表示

## 技術構成

- Web/PWA: 初期MVP
- QGIS: GISデータ編集
- MapLibre GL: Web地図
- GitHub Pages: 配信
- Googleスプレッドシート + Apps Script: 訓練用の簡易同期
- Native app: iPhoneバックグラウンド測位が必須になった場合の次段階
